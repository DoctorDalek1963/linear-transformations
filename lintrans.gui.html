This module provides the LintransMainWindow class, which provides the main window for the GUI.

class lintrans.gui.main_window.LintransMainWindow

Bases: PyQt5.QtWidgets.QMainWindow

This class provides a main window for the GUI using the Qt framework.

This class should not be used directly, instead call lintrans.gui.main_window.main() to create the GUI.

__init__()

Create the main window object, and create and arrange every widget in it.

This doesn’t show the window, it just constructs it. Use lintrans.gui.main_window.main() to show the GUI.

dialog_define_matrix(dialog_class: Type[lintrans.gui.dialogs.define_new_matrix.DefineDialog]) → None

Open a generic definition dialog to define a new matrix.

The class for the desired dialog is passed as an argument. We create an instance of this class and the dialog is opened asynchronously and modally (meaning it blocks interaction with the main window) with the proper method connected to the dialog.finished slot.

Note

dialog_class must subclass lintrans.gui.dialogs.define_new_matrix.DefineDialog.

dialog_class (Type[lintrans.gui.dialogs.define_new_matrix.DefineDialog]) – The dialog class to instantiate
